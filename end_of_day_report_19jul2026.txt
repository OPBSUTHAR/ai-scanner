PROJECT STRUCTURE NOTE
----------------------
There are TWO data/ folders in this project:

  1. ai_scanner/data/        ← ACTIVE — used at runtime by all entry points
     LocalStorage(base_dir="data") resolves here because all launchers run from
     inside ai_scanner/. Contains real documents, metadata JSONs, uploads, PDFs.
     The .gitignore has "data/*" so only .gitkeep is tracked.

  2. (root)/data/            ← DEAD / VESTIGIAL
     Empty subdirectories (documents/, metadata/, temp/, uploads/). Not referenced
     by any code anywhere in the project. Safe to delete.

  The root-level generate_project.py created ai_scanner/data/.gitkeep during initial
  scaffolding. The subdirectories are created dynamically by LocalStorage._ensure_dirs().
